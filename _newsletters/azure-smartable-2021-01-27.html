Generally available: Copy Blob support over private endpoints in Azure Storage Azure Storage now supports copying of data between storage accounts using private endpoints. This includes support for the Copy Blob API and utilities such as AzCopy.

Public preview: Azure Key Vault CSI driver support in Azure Kubernetes Service Azure Key Vault provider for Secrets Store CSI driver allows you to get secret contents stored in an Azure Key Vault instance. Azure Key Vault simplifies the task of securing and using cryptographic keys and other secrets with AKS. With this new feature, you can use the Secrets Store CSI driver interface
